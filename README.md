# 🎥 Video Downloader Pro - Frontend

Beautiful React frontend for downloading videos from 20+ platforms.

## 🚀 Quick Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone)

## ✨ Features

- 🎬 Download from YouTube, Instagram, Facebook, TikTok & more
- 🎨 Modern UI with Tailwind CSS
- 📱 Fully responsive design
- ⚡ Fast & lightweight with Vite
- 🔒 No watermarks, HD quality

## 🌐 Live Demo

Backend API: https://video-downloader-backend-c90x.onrender.com

## 🛠️ Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons

## 📦 Local Development

```bash
npm install
npm run dev
```

## 🚀 Deploy to Vercel

```bash
npm install -g vercel
vercel login
vercel
```

Or push to GitHub and import on Vercel dashboard.
